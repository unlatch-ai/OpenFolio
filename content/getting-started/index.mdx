# Getting Started

This guide walks you through setting up OpenFolio for local development.

## Prerequisites

- **Node.js** 18.17.0 or later
- **pnpm** — install via `npm install -g pnpm` or `corepack enable`
- **Docker** — for local Supabase (PostgreSQL + Auth)
- **OpenAI API key** — for embeddings and AI features

## Quick Start

```bash
# Clone the repository
git clone https://github.com/unlatch-ai/OpenFolio.git
cd openfolio

# Install dependencies
pnpm install

# Start local Supabase (requires Docker)
pnpm supabase:start

# Copy environment template
cp .example.env.local .env.local

# Seed the local database with a dev user
pnpm db:seed:local

# Start the dev server
pnpm dev
```

Visit [http://localhost:3000](http://localhost:3000) and log in with:
- **Email:** `dev@local.test`
- **Password:** `devpassword`

## Available Commands

| Command | Description |
|---------|-------------|
| `pnpm dev` | Start development server |
| `pnpm build` | Build for production |
| `pnpm test` | Run test suite |
| `pnpm lint` | Run ESLint |
| `pnpm supabase:start` | Start local Supabase |
| `pnpm supabase:stop` | Stop local Supabase |
| `pnpm db:reset` | Reset local DB (migrations + seed) |
| `pnpm db:seed:local` | Create dev user + workspace |
| `pnpm db:seed:embeddings` | Generate embeddings for seed data |
| `pnpm db:types:local` | Regenerate TypeScript types from schema |
