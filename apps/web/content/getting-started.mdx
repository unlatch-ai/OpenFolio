# Getting Started

## Install

1. Download the latest macOS release from GitHub Releases.
2. Move `OpenFolio.app` into `/Applications`.
3. Launch the app.
4. Grant Full Disk Access so OpenFolio can read `~/Library/Messages/chat.db`.
5. Import Messages and begin searching.

## Account mode

You can use OpenFolio locally without signing in.

Sign in only when you want hosted features such as:

- billing-backed entitlements
- hosted AI
- managed connectors
- future hosted MCP / remote access

## Optional BYOK AI

If you provide your own OpenAI key, OpenFolio can generate local embeddings and answers without requiring a paid hosted plan.
